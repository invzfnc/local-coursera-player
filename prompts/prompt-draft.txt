I would like to bulid a coursera-like UI for playing local downloaded coursera videos. It should have these basic functions: play videos, sort and list videos by week and topics, keep track of learning progress.

Some requirements for this project:
- should be web-based. use JavaScript, HTML, and CSS only.
- index.html as the starting point.
- no dependencies, no installation, portable.
- efficient and small sized project.
- for personal use only. non commercial, no enterprise level stuff.
- no login. no account management. keep it simple.
- user specifies a folder path. program should search the folder recursively to list all videos in the sidebar (video folder structure will be provided shortly)
- has a settings page that supports tweaks such as:
    - setting source video folder path
    - switch themes: dark theme or light theme
    - video seeking preference: 5 seconds or 10 seconds? etc
- keep stored data at local folder. create a data/ folder to keep all the info needed (example: config.js or settings.js for keeping the settings, progress.js to keep track of learning progress, or whatever deemed fit)
- for the video player: make it like the YouTube video player. Supports subtitles (can be fetched from the srt file found in the provided folder), video playback speed adjustments (0.75x, 1x, 1.25x, 1.5x, 1.75x, 2x), long press spacebar to fast forward 2x speed (back to original speed when released, exactly like YouTube)
- go to next item button on the bottom right

exclude these functions/do not implement these functions:
- daily goals/motivation
- start free trial button
- account management/login feature
